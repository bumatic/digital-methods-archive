Code is beautiful!

At QuantifiedCode, we help development teams write better code—through data. On our free code analysis platform, we use visualizations, to make code better understandable for developers and business people. Creating great visualizations can be challenging though. With this project, we want to change that by providing an open-source collection of useful and engaging source code visualizations. They are all realized with either D3 or Three.js and are provided under an Affero GPL license. You can freely reuse them in your own projects. If you like to see a visualization of your own project, check out our free service.

Please feel free to contribute your own visualizations.
