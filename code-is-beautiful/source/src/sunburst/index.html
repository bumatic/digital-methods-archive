The sunburst chart is a hierarchical pie chart which maps the directory structure of the project to a set of circles.

The size of each segment (in degrees) is given by the number of lines in a given directory / file. The color of each segment can be used to represent various metrics, e.g. the average cyclomatic complexity per line of code.
